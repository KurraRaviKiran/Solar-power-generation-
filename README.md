# Solar Power Generation Dashboard Analysis

## Project Overview
Analyzed real-world solar power generation data across 12 months 
and 3 production sources (Solar, Wind, Mixed) to identify trends 
and seasonal patterns.

## Key Metrics
- Total Production: 322M units
- Avg Daily Production: 149.16K units
- Peak Daily Production: 476K units
- Lowest Daily Production: 26K units
- Target: 645M units

## Key Insights
- Identified 18x seasonal variance between peak and lowest production
- Summer season contributed highest output (27.93%)
- Wind source outperformed Solar in total generation

## KPIs Tracked
- Total Production
- Average Daily Production
- Peak Daily Production
- Lowest Daily Production
- Target Achievement Rate (322M vs 645M)
